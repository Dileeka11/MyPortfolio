# Magic Web Gallery

## Project info

This is a web gallery application built with React, Vite, and Tailwind CSS.

## Getting Started

To get started with development:

1.  **Clone the repository:**

    ```sh
    git clone <YOUR_GIT_URL>
    ```

2.  **Navigate to the project directory:**

    ```sh
    cd magic-web-gallery
    ```

3.  **Install dependencies:**

    ```sh
    npm install
    ```

4.  **Start the development server:**
    ```sh
    npm run dev
    ```

## Technologies Used

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS
